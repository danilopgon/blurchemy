# Roadmap

## Foundation

- Angular CLI 22 app shell with pnpm.
- Strict TypeScript configuration.
- Tailwind CSS 4 through the Angular CLI stylesheet pipeline.
- Vitest unit testing through Angular's test builder.
- ESLint and Prettier tooling.

## Next

- Follow the three-spec SDD plan in `docs/sdd-spec-plan.md`.
- Define the SDD proposal/spec/design for the first Blurchemy editor increment.
- Introduce browser-only image import and preview.
- Add focused tests for any pure utility or processing boundary introduced.
- Define preset data contracts before building preset UI.

## Later

- Improve blur and motion blur quality.
- Add render cancellation and progress states.
- Add keyboard-accessible editing shortcuts.
- Move heavy rendering to Web Workers.
- Add local preset authoring and import/export of preset JSON.
- Add layered effects: bloom, halation, dust, paper, and chromatic drift.
- Add history snapshots and compare modes.
